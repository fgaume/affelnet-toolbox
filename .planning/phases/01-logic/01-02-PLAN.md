<feature>
  <name>Affelnet Score Calculation</name>
  <files>src/services/scoreCalculation.ts, src/services/__tests__/scoreCalculation.test.ts</files>
  <behavior>
    - Subject averaging: Handles grouping 12 subjects into 7 disciplinary fields.
    - Fallback: EMC uses Histoire-Géo value if null.
    - Harmonization: Applies H = 10 * (10 + (T - mu) / sigma) to each field.
    - Weighting: Applies 5x for FR/Math, 4x for others.
    - Final calculation: Sum of weighted harmonized scores * 2.
    - Edge cases: Handles missing grades (null values) by excluding them from averages within a field, but harmonized result remains 0 if field has no grades.
  </behavior>
  <implementation>
    Create a function `calculateScore(grades: UserGrades, stats: Record<DisciplinaryField, AcademicStats>): UserScore`.
    The function must iterate through the 7 fields, calculate raw averages, apply harmonization, apply weights, and sum them up.
  </implementation>
</feature>
